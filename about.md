## About This Blog

This blog is now powered by **Jekyll** and hosted on **GitHub Pages**.

### Features

- ✅ **完全自动化** - 写完文章推送即可，无需手动维护索引
- ✅ **分类管理** - 按主题分类，方便查找
- ✅ **响应式设计** - 适配 PC 和移动端
- ✅ **Markdown 支持** - 使用 Markdown 编写文章
- ✅ **SEO 优化** - 自动生成 sitemap 和 RSS feed

### How to Add New Posts

1. 在 `_posts/` 目录下创建新的 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-title.md`
3. 在文件头部添加 YAML Front Matter：

```yaml
---
layout: post
title: "文章标题"
date: YYYY-MM-DD
categories: dba  # 或 devops, biopharma, enterprise, notes, csv
author: Lifu
---
```

4. 推送到 GitHub，自动部署！
